# Day 6 – AI Resume Optimization

## Objective

Learn how AI can analyze and optimize a resume for Applicant Tracking Systems (ATS) by identifying weaknesses, improving formatting, and enhancing content for better job opportunities.

---

# Task Performed

Today I used Claude as an AI Resume Optimizer.

The process included:

1. Uploading my existing resume.
2. Using the provided Resume Optimizer prompt.
3. Reviewing the ATS analysis generated by Claude.
4. Identifying missing keywords and improvement areas.
5. Generating an optimized one-page resume.
6. Saving the optimized version for future internship applications.

---

# Resume Analysis

Claude evaluated my resume based on:

- ATS compatibility
- Resume structure
- Technical skills
- Project descriptions
- Experience section
- Keyword optimization
- Formatting consistency
- Overall readability

---

# Improvements Suggested

Some of the recommendations included:

- Improve ATS-friendly formatting.
- Strengthen project descriptions using action verbs.
- Add measurable achievements where possible.
- Include relevant technical keywords.
- Organize skills into clear categories.
- Keep the resume concise and limited to one page.

---

# Optimized Resume

Claude generated an improved one-page resume with:

- Better formatting
- Clear section hierarchy
- ATS-friendly keyword optimization
- Stronger project descriptions
- Improved readability
- Professional presentation

---

# Key Learnings

- ATS systems scan resumes for keywords before recruiters review them.
- A clean, structured resume increases readability.
- Quantifying achievements makes projects more impactful.
- AI can quickly identify areas for improvement and suggest practical enhancements.

---

# My Reflection

Today's exercise showed me that creating a good resume is not only about listing skills and projects but also about presenting them in a way that is optimized for both ATS software and recruiters. AI helped identify improvements that made my resume more professional and internship-ready.
