# Day 16 – Claude Custom Skill: Stock Fundamental Research

## Objective

Learn how to create and use a reusable Custom Skill in Claude for performing stock fundamental research. The skill analyzes financial performance, valuation, ownership trends, risk factors, and provides investment insights without requiring the full prompt every time.

---

# Task Performed

Today I created a reusable Custom Skill in Claude named **stock-fundamental-research**.

The workflow included:

1. Creating a new Custom Skill.
2. Naming the skill **stock-fundamental-research**.
3. Adding the provided description.
4. Adding the provided instructions.
5. Saving the Custom Skill.
6. Testing the skill using a stock (e.g., TCS, Infosys, Reliance, HDFC Bank, or Tata Motors).
7. Generating a complete stock research report.
8. Reviewing valuation analysis, ownership trends, financial performance, and risk factors.
9. Capturing screenshots of the created skill and generated report.
10. Verifying that the Custom Skill can be reused without entering the original prompt again.

# Analysis Performed

The generated report included:

# Key Learnings

- Custom Skills allow prompts to be reused efficiently.
- A well-designed skill saves time and ensures consistent outputs.
- AI can quickly generate structured stock research reports.
- Financial analysis becomes easier with reusable workflows.
- Custom Skills improve productivity for repetitive tasks.

---

# My Reflection

Today's activity demonstrated how Claude Custom Skills can automate recurring tasks. Instead of rewriting a detailed prompt every time, the saved skill generated a comprehensive stock research report with consistent structure and quality. This makes Custom Skills valuable for research, analysis, and professional workflows.
